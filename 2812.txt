크게 만들기 

문제
{
N자리 숫자가 주어졌을 때, 여기서 숫자 K개를 지워서 얻을 수 있는 가장 큰 수를 구하는 프로그램을 작성하시오.
}

입력
{
첫째 줄에 N과 K가 주어진다. (1 ≤ K < N ≤ 500,000)

둘째 줄에 N자리 숫자가 주어진다. 이 수는 0으로 시작하지 않는다.
}

출력
{
입력으로 주어진 숫자에서 K개를 지웠을 때 얻을 수 있는 가장 큰 수를 출력한다.
}

예제 입력 1 
{
4 2
1924
}

예제 출력 1 
{
94
}

예제 입력 2 
{
7 3
1231234
}

예제 출력 2 
{
3234
}

예제 입력 3 
{
10 4
4177252841
}

예제 출력 3
{ 
775841
}

풀이 방법
{
    그리디 + 스택 활용

    n-k개를 선택하여 가장 큰 숫자를 만드는 것으로 생각하고 문제를 풀었다.

    스택을 이용하여 숫자를 담을것인데 bottom에는 가장 큰 값이 들어가고 top에는 가장 작은 값이 들어가도록 설정한다.

    위의 스택을 채우기 위해서 처음부터 끝까지 숫자를 top과 비교하면서 push, pop을 반복하되, 현재 비교하는 숫자 뒤에

    남아있는 숫자의 개수가 채워야하는 숫자의 개수보다 많은 경우에만 push와 pop을 진행하고 없으면 그자리에서 stack을 출력하고 종료한다.

    만약, stack의 길이가 굉장히 길어지게 되면 마지막 까지 숫자를 채워놓고 stack[:n-k]를 출력하면 된다.
}
