비슷한 단어

문제
{
N개의 영단어들이 주어졌을 때, 가장 비슷한 두 단어를 구해내는 프로그램을 작성하시오.

두 단어의 비슷한 정도는 두 단어의 접두사의 길이로 측정한다. 접두사란 두 단어의 앞부분에서 공통적으로 나타나는 부분문자열을 말한다. 

즉, 두 단어의 앞에서부터 M개의 글자들이 같으면서 M이 최대인 경우를 구하는 것이다. 

"AHEHHEH", "AHAHEH"의 접두사는 "AH"가 되고, "AB", "CD"의 접두사는 ""(길이가 0)이 된다.

접두사의 길이가 최대인 경우가 여러 개일 때에는 입력되는 순서대로 제일 앞쪽에 있는 단어를 답으로 한다. 

즉, 답으로 S라는 문자열과 T라는 문자열을 출력한다고 했을 때, 우선 S가 입력되는 순서대로 제일 앞쪽에 있는 단어인 경우를 출력하고, 

그런 경우도 여러 개 있을 때에는 그 중에서 T가 입력되는 순서대로 제일 앞쪽에 있는 단어인 경우를 출력한다.
}

입력
{
첫째 줄에 N(2 ≤ N ≤ 20,000)이 주어진다. 다음 N개의 줄에 알파벳 소문자로만 이루어진 길이 100자 이하의 서로 다른 영단어가 주어진다.
}

출력
{
첫째 줄에 S를, 둘째 줄에 T를 출력한다. 단, 이 두 단어는 서로 달라야 한다. 즉, 가장 비슷한 두 단어를 구할 때 같은 단어는 제외하는 것이다.
}

예제 입력 1 
{
9
noon
is
lunch
for
most
noone
waits
until
two
}

예제 출력 1 
{
noon
noone
}

예제 입력 2 
{
4
abcd
abe
abc
abchldp
}

예제 출력 2 
{
abcd
abc
}

풀이 방법
{
    접두사를 기준으로 같은 것을 판단하기 때문에 dict()를 이용하여

    주어진 문자열들을 사전의 순서에 따라 정렬시켜 준 뒤 바로 뒷 글자와의 접두사 길이를 체크하여

    저장하고 최대 길이를 구하면 갖는 문자열을 찾으면 된다.

    이 문제의 핵심 포인트는 사전의 순서에 따라서 문자열을 정렬하는 것이다.

    사전 순저에 따라 정렬하게 되면 접두사 비교는 정렬된 문자열 배열에서 인접한 2개의 문자열만 진행하면 된다.

    위의 방식대로 진행한 후, 각 문자열의 접두사 길이를 배열에 넣어둔다.

    이후, 앞에서부터 문자열을 탐색하며 접두사의 길이가 가장 긴 값과 같은 문자열을 찾으면 출력하고 
    
    이후에 동일한 접두사를 갖고 있는 문자열을 탐색하여 출력하면 문제를 해결할 수 있다.
}

느낀점
{
    생각보다 실수를 많이 한 문제이다.

    문제 자체는 어렵지 않았는데 if 문과 break를 적절하게 사용하지 못해서 여러번 문제를 틀렸던 것 같다.

    무작정 반례를 찾아 돌리는 것도 좋지만 틀렸다면 차분하게 코드를 다시 보는 것이 중요한 것 같다.
}
