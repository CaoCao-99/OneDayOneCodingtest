부분수열의 합 2


문제
{
N개의 정수로 이루어진 수열이 있을 때, 크기가 양수인 부분수열 중에서 그 수열의 원소를 다 더한 값이 S가 되는 경우의 수를 구하는 프로그램을 작성하시오.
}


입력
{
첫째 줄에 정수의 개수를 나타내는 N과 정수 S가 주어진다. (1 ≤ N ≤ 40, |S| ≤ 1,000,000) 둘째 줄에 N개의 정수가 빈 칸을 사이에 두고 주어진다. 주어지는 정수의 절댓값은 100,000을 넘지 않는다.
}


출력
{
첫째 줄에 합이 S가 되는 부분수열의 개수를 출력한다.
}


예제 입력 1 
{
5 0
-7 -3 -2 5 8
}

예제 출력 1 
{
1
}

풀이 방법
{
    수열을 정렬하여 반으로 나누고 좌측의 (최대 20개의) 모든 가능 경우의 수(합)을 저장한다.

    우측도(최대 20개) 모든 가능 경우의 수(합)을 저장한다.

    이후, 좌측에서 현재 만들고자 하는 값이 있는 경우 cnt를 늘려주고 우측도 마찬가지로 한번 돌린다.

    마지막으로 좌측에서 값을 선택하고 만들어야 하는 값의 차이만큼 우측에서 구할 수 있는 지 이진탐색을 이용하여 문제를 해결한다.
}
