고층 건물

문제
{
세준시에는 고층 빌딩이 많다. 

세준시의 서민 김지민은 가장 많은 고층 빌딩이 보이는 고층 빌딩을 찾으려고 한다. 

빌딩은 총 N개가 있는데, 빌딩은 선분으로 나타낸다. i번째 빌딩 (1부터 시작)은 (i,0)부터 (i,높이)의 선분으로 나타낼 수 있다. 

고층 빌딩 A에서 다른 고층 빌딩 B가 볼 수 있는 빌딩이 되려면, 두 지붕을 잇는 선분이 A와 B를 제외한 다른 고층 빌딩을 지나거나 접하지 않아야 한다. 

가장 많은 고층 빌딩이 보이는 빌딩을 구하고, 거기서 보이는 빌딩의 수를 출력하는 프로그램을 작성하시오.
}

입력
{
첫째 줄에 빌딩의 수 N이 주어진다. N은 50보다 작거나 같은 자연수이다. 둘째 줄에 1번 빌딩부터 그 높이가 주어진다. 높이는 1,000,000,000보다 작거나 같은 자연수이다.
}


출력
{
첫째 줄에 문제의 정답을 출력한다.
}


예제 입력 1 
{
15
1 5 3 2 6 3 2 6 4 2 5 7 3 1 5
}

예제 출력 1 
{
7
}

예제 입력 2 
{
1
10
}

예제 출력 2 
{
    0
}


예제 입력 3 
{
4
5 5 5 5
}

예제 출력 3 
{
2
}

예제 입력 4 
{
5
1 2 7 3 2
}

예제 출력 4 
{
4
}

예제 입력 5 
{
10
1000000000 999999999 999999998 999999997 999999996 1 2 3 4 5
}

예제 출력 5 
{
6
}

풀이 방법1(실패)
{
    이전에 풀었던 빌딩 문제와 똑같이 풀면 될 것 같다.

    2가지 값을 구하면 문제를 해결할 수 있다.

    1. 현재 건물을 기준으로 왼쪽의 건물이 보이는 경우 중 큰 경우(본인의 건물보다 작은 건물들을 보는 경우, 본인의 건물보다 큰 건물들을 보는 경우)

    2. 현재 건물을 기준으로 오른쪽의 건물이 보이는 경우 중 큰 경우(본인의 건물보다 작은 건물들을 보는 경우, 본인의 건물보다 큰 건물들을 보는 경우)

    위의 stack배열 총 4개를 이용하여 좌측(2개 중 큰 값) + 우측(2개 중 큰 값)으로 각 빌딩에서 볼 수 있는 건물의 수를 구할 수 있고

    이 배열에서 가장 큰 값을 출력하면 된다.
}

풀이 방법2
{
    이전에 풀었던 문제와는 확실히 다른 문제였다.

    문제를 제대로 이해하지 못하고 풀었기에 위와 같은 풀이를 생각했다.

    문제를 풀기위해서는 우선 큰 틀에서 문제를 접근해야 한다.

    현재 건물을 기준으로 기울기를 따져서 문제를 풀어야 하는데 좌측으로 갈때는 기울기의 값이 점점 작아져야 하고

    우측으로 갈때, 기울기가 점점 커지기만 하면 문제를 해결할 수 있다.

    현재 건물을 기준으로 좌 우측에 포물선을 그려서 생각해보면 쉽다.

    따라서, 현재 건물에 좌측과 우측을 for 구문을 이용하여 기울기를 변화시켜주어 값을 비교하여 좌측은 기울기가 더 작아지면 개수를 늘려주고 기울기가 늘어나는 경우는 넘어가면 된다.

    우측도 마찬가지로 진행하면 문제를 해결할 수 있다.
}

느낀 점
{
    이전에 풀었던 문제와 유사하다고 느껴서 바로 접근한 것이 문제였다.

    앞으로는 입력 예제와 출력 예제를 잘 보고 문제를 풀어 보자

    이 문제도 다시 풀어 보자!
}
