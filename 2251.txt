물통 다국어

문제
{
각각 부피가 A, B, C(1≤A, B, C≤200) 리터인 세 개의 물통이 있다. 처음에는 앞의 두 물통은 비어 있고, 세 번째 물통은 가득(C 리터) 차 있다. 

이제 어떤 물통에 들어있는 물을 다른 물통으로 쏟아 부을 수 있는데, 이때에는 한 물통이 비거나, 다른 한 물통이 가득 찰 때까지 물을 부을 수 있다. 

이 과정에서 손실되는 물은 없다고 가정한다.

이와 같은 과정을 거치다보면 세 번째 물통(용량이 C인)에 담겨있는 물의 양이 변할 수도 있다. 

첫 번째 물통(용량이 A인)이 비어 있을 때, 세 번째 물통(용량이 C인)에 담겨있을 수 있는 물의 양을 모두 구해내는 프로그램을 작성하시오.
}

입력
{
첫째 줄에 세 정수 A, B, C가 주어진다.
}

출력
{
첫째 줄에 공백으로 구분하여 답을 출력한다. 각 용량은 오름차순으로 정렬한다.
}

예제 입력 1 
{
8 9 10
}

예제 출력 1 
{
    1 2 8 9 10
}

풀이 방법
{
    BFS와 딕셔너리를 사용하면 문제를 해결할 수 있다.

    A, B, C 물통 중 무조건 하나의 물통은 비어 있다.

    나머지 물통에 물이 들어있다면 위의 규칙을 이용하여 BFS를 돌려주면 된다.

    만약, 이미 이전과 같은 상황이 존재 했다면(dict 에 값이 존재하는 경우)queue에 넣지 않는다.

    그렇지 않으면 queue에 값을 넣어 반복문을 돌려준다.

    위와 같은 방식으로 queue가 빌 때까지 돌게 되면 문제를 해결할 수 있다.

    정답 또한, 모든 경우의 값만 적으면 되기 때문에 set()를 이용하여 중복 없이 정렬하면 정답을 도출할 수 있다.
}
