강의실 배정

문제
{
수강신청의 마스터 김종혜 선생님에게 새로운 과제가 주어졌다. 

김종혜 선생님한테는 Si에 시작해서 Ti에 끝나는 N개의 수업이 주어지는데, 최소의 강의실을 사용해서 모든 수업을 가능하게 해야 한다. 

참고로, 수업이 끝난 직후에 다음 수업을 시작할 수 있다. (즉, Ti ≤ Sj 일 경우 i 수업과 j 수업은 같이 들을 수 있다.)

수강신청 대충한 게 찔리면, 선생님을 도와드리자!
}


입력
{
첫 번째 줄에 N이 주어진다. (1 ≤ N ≤ 200,000)

이후 N개의 줄에 Si, Ti가 주어진다. (0 ≤ Si < Ti ≤ 109)
}

출력
{
강의실의 개수를 출력하라.
}

예제 입력 1 
{
3
1 3
2 4
3 5
}


예제 출력 1 
{
    2
}

풀이 방법
{
    우선순위 큐를 이용하여 문제를 해결할 수 있다.

    일단, 주어지는 n의 크기가 200000이기 때문에 시간복잡도 O(NlogN)도 넘지 않는다.

    우선순위 큐를 2개 생성하면 문제를 해결할 수 있다.

    첫 번째 우선순위 큐는 시작 시간을 담고 있는 우선순위 큐다.

    두 번째 우선순위 큐는 끝나는 시간을 담고 있는 큐다.

    첫 번째 큐는 아직 강의실을 배정하지 않은 수업을 담은 큐고

    두 번째 큐는 현재 강의실을 배정받아 수업하고 있는 큐다.

    따라서, 첫 번째 우선순위 큐에서 시작 시간이 가장 짧은 수업과 두 번째 큐에서 종료시간이 가장 짧은 수업을 비교해 

    시작 시간이 두 번째 우선순위 큐에서 나온 종료시간보다 뒤라면 첫 번째 우선순위 큐에서 나온 수업의 종료시간을 두 번째 큐에 넣어주면 된다.

    만약, 시작 시간이 종료시간보다 앞이라면 본인의 수업 종료시간을 두 번째 우선순위에 넣어주면 된다.

    위의 과정을 첫 번째 우선순위 큐가 빌때까지 즉, 모든 수업에 강의실 배정이 완료될 때 까지 진행해주면 두 번째 우선순위 큐의 길이가 

    강의실 개수가 된다. 

    결과적으로 2개의 우선순위 큐를 이용하여 문제를 해결할 수 있다.
}
